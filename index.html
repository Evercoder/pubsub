Pub/Sub

pubsub is a small, fast JavaScript library that implements the Publish/Subscribe pattern for web applications. You can read through the annotated source code.

In a nutshell


var ps = new pubsub();

// subscribe to an event
ps.sub('hello', function(person) {
	console.log('Hi, ' + person + '!');
});

// publish an event
ps.pub('hello', 'Dan');

More features:

  • namespaced events, e.g. namespace:event;
  • subscribe to events were triggered prior to the subscription with pubsub.recoup;
  • execute a subscribed function only once, then unsubscribe with pubsub.once;
  • it's fast.
